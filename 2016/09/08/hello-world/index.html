酷龙——7年博主，重新归来

发表于 2016-09-08

之前一直有一个博客（2009~2014）而这段时间也是博客的黄金时代，也是互联网淘金的风口时代。很荣幸我也被风给吹了起来。5年差不多写了700篇文章，录制了接近200个视频。这也给我带来了不菲的收入，进入了培训的行业。后来因为个人及国家政策等原因，我在我熟悉，我喜欢的行业跌倒了。导致博客及论坛停更了两年。

之所以你能看到这个博客是目前我正在用GIT（版本管理软件）做一个项目。无意中发现了 hexo （静态博客生成框架）第一印象是被它的简约阅读体验给打动了。折腾了好几天，成了现在这个样子。慢慢发现 hexo 真的没有宣传的那么简单，特别是换电脑写作这方面。等有时间，我把我现在用的主题，做一个 WordPress 的主题，提供给大家用。

现在总结一下，这几年：

  • 09年，建立资源站，卖VIP赚钱（资源全手工整理）
  • 10年，做 Google AdWords 培训，开创分类信息竞价
  • 11年，开始做网络营销培训
  • 12年，新增中小企业营销培训，电商培训。
  • 13年，互联网知名培训机构，培训内容包含成功学，营销，及产品开发。
  • 14年，业务一天天的下滑，直到交不起房租。
  • 15年，帮家里打理生意（其实就是当司机）
  • 16年，在一家集团，做电商（女鞋，音响）

14年到16年是我个人知识储备多的主要是3个方面

  1. 应用软件方面
  2. Linux 运维方面
  3. Linux 嵌入式开发
  4. 摄影

应用软件方面，主要是工业设计这块。Rhinoceros, Keyshot,野火等3D及工业应用软件。PS是我老本行，对AI有重新的认识。

Linux 运维方面，因为个人兴趣之前就对Linux很熟悉，但是没有运维过大的项目。在实际的高可用方面，经验不足。但是最近这几年，认识了这方面的很多朋友，也经常免费给企业解决一些小问题。

Linux 嵌入式开发，是因为到时老罗做手机，我一时兴起就学了。现在，偶尔看看淘宝有没有什么新的好玩的项目，买回来玩玩。

摄影，主要局限于静物产品拍摄方面。

目前在一家公司上班，说是说五天八小时，实际上周一到周五经常加班。一直以来，我都没有放弃培训这个行业。互联网无时不刻都在变化，希望我能将我的价值分享出去。
